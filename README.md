## What I'm building

**[DuckyCut](https://github.com/PedroRamosPereira/duckycut)** — CEP extension for Adobe Premiere Pro. Detects silences via FFmpeg and auto-generates FCP7 XML cut timelines. Inspired by FireCut, built from scratch.
